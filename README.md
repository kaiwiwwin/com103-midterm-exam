# COM103 Midterm Exam - Expense Tracker

## Description
This program is a simple expense tracker that allows a student to:
- Input weekly budget
- Record up to 4 expenses
- Categorize each expense
- Detect high expenses (over 25% of budget)
- Display total spent, remaining balance, and status

## How to Run
1. Make sure Python is installed
2. Run the program using:
   python midterm_solution.py
